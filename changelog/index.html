Changelog

YearMap release history, followed by earlier project milestones.

Version 1.0.4 — Read-only calendar sharing

June 24, 2026

  • Added local-first share links for publishing a read-only snapshot of your YearMap.
  • Added a shared calendar viewer with range details, notes, current-day highlighting, and a link to create your own YearMap.
  • Expanded the Range details editor so ranges can be moved to another category or deleted directly.
  • Improved range tooltip accuracy after loading or editing calendar data.
  • Added clearer confirmation when a share link has been copied.

Version 1.0.3 — Same-category overlapping ranges

June 21, 2026

  • Added support for independent, overlapping ranges within the same category.
  • Added a chooser for editing or deleting the exact range when several ranges share a date.
  • Added subtle tonal bands to reveal overlaps while preserving the existing category-lane layout.
  • Kept category totals accurate by counting unique assigned days.

Version 1.0.2 — Range details editor

June 17, 2026

  • Added a sidebar Range details editor for changing a range’s start date, end date, and note in one place.
  • Added an optional setting to focus the note field immediately after assigning a range for faster note entry.
  • Improved undo behavior for assignments, edits, and deletions.
  • Updated the website with a short product video and refreshed interface screenshots.

Version 1.0.1 — Calendar access improvements

June 9, 2026

  • Improved reliable access to YearMap from its toolbar icon.
  • Clarified how YearMap behaves when it or another extension controls Chrome’s new-tab page.
  • Removed an ineffective new-tab replacement preference.

Version 1.0.0 — Chrome Web Store launch

June 7, 2026

  • YearMap became available on the Chrome Web Store.
  • Updated the site with install links, support pages, privacy policy, and terms.
